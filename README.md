# mmci_Generator_Tool 開發與使用說明

## 檔案資訊
- 檔案名稱：`mmci_Generator_Tool.v1.0.1.rar`
- 用途：**FW 快速打包工具 (FW Fast Package Tool)**
- 適用系統：Windows 10 22H2 以上

---

## 使用方式

### 模式選擇
- USB 模式  
- PC ULD 模式  

### 操作說明
- 單選模式  
  - 拖曳 **資料夾** 至程式空白處，自動排列並抓取 `.mci`  
- 多選模式  
  - **SHIFT + 左鍵** → 依 Windows 資料顯示順序排列  
  - **CTRL + 左鍵** → 依 Windows 資料顯示順序排列  
- 指定輸出資料夾  
- 點擊 **生成**  
  - 追蹤 `.mci` 生成完成  
  - 輸出對應 `.mmci` 與同名資料夾
